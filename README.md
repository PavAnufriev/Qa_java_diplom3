# QA Java Диплом 3

Дипломный проект по курсу "Автоматизация тестирования"

## Структура проекта

```
qa-java-diplom3/
├── src/
│   ├── main/
│   │   ├── java/
│   │   └── resources/
│   └── test/
│       ├── java/
│       └── resources/
├── pom.xml
├── .gitignore
└── README.md
```

## Технологии

- Java 11
- Maven
- JUnit 5

## Запуск тестов

```bash
mvn test
```

## Сборка проекта

```bash
mvn clean compile
```
