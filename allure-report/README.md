# Allure отчет о тестировании

## Как просмотреть отчет

### Способ 1: Открыть статический HTML отчет
Откройте файл `index.html` в браузере для просмотра полного отчета.

### Способ 2: Запустить локальный сервер
```bash
# Перейти в корень проекта
cd path/to/project

# Запустить Allure сервер
mvn allure:serve
```
